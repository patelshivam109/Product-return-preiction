# Return Risk Prediction System

## Overview
Machine Learning solution that predicts product return risk and identifies high-risk orders.

## Features
- EDA & Business Insights
- Feature Engineering
- Logistic Regression
- Random Forest
- XGBoost
- Hyperparameter Tuning
- Risk Scoring
- High-Risk Order Detection
- Interactive Streamlit Dashboard
